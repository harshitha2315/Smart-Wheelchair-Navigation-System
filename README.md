# Smart Wheelchair Navigation System

## Overview
An IoT-enabled smart wheelchair designed to improve mobility, safety, and independence for differently-abled individuals.

The system combines obstacle detection, GPS tracking, and mobile app control to provide safer navigation and real-time monitoring.

## Features
- Obstacle detection and avoidance
- GPS location tracking
- Mobile app-based wheelchair control
- Real-time monitoring
- Enhanced user safety

## Technologies Used
- ESP8266 / NodeMCU
- GPS Module (NEO-6M)
- Ultrasonic Sensor
- Motor Driver
- IoT Platform
- Mobile Application
